# OMSCS
Code: https://github.com/ghanley/OMSCS/ML/Assignment2

nnweights.py will run SA, GA, RHC, and GD neural nets and output charts on accuracy and timings for each.

tsp.py, kcolor.py, and 6peaks.py will run the three optimization problems and output the fitness and timings charts for each algorithm used in paper. 

The BreastCancer data set was obtained from https://www.openml.org/d/1510 and needs to be saved in the folder with the learners, currently as "breast.csv".
 

Libraries needed to run code:
sklearn, 
matplotlib, 
pandas, 
numpy,
mlrose
